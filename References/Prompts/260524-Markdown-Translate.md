# 任务：翻译 Markdown 文件（创建翻译 Skill）

## 输入
用户会指定一个 Markdown 文件或文件夹，将其中的英文内容翻译为中文。

---

## 翻译规则

### 需要翻译的内容
- 正文中成段的文字
- Markdown 表格中的单元格内容

### 不需要翻译的内容
- 文件开头的目录（Table of Contents）
- 文件结尾的参考文献列表（References）
- 论文引用标记，例如 `\[Xu26e\]`、`[Xu26e]` 等格式的 key
- 专业术语、技术名词（保留英文原文）

---

## 格式要求

1. **原文只读**：严禁删除、修改或改动原文的任何内容。
2. **逐段对照**：对每一段正文，将译文紧接在原文段落之后插入，形成英中逐段对照的格式。
3. **表格翻译**：将整个表格翻译为中文后，作为一个完整的新表格紧接在原始表格之后插入。
4. **不改变文档结构**：不得新增、删除或调整原有标题、章节、列表等结构。

---

## 输出与保存

### 译文文件
- **保存位置**：译文文件保存在与原文件相同的目录下。
- **命名规则**：`{原文件名}_zh_cn.md`
  - 示例：`overview.md` → `overview_zh_cn.md`
- 如果输入为文件夹，则对文件夹内所有 Markdown 文件执行上述操作。
